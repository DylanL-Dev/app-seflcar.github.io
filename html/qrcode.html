Voici la façon dont il fonctionne :

  1. Ouvrez une application de scanner QR code sur votre téléphone
  2. Lancez l'application de scanner QR code et autorisez l'accès à la caméra si nécessaire.
  3. Dirigez la caméra vers le QR code imprimé ou affiché à l'écran.
  4. Assurez-vous que le QR code est clairement visible dans le cadre de la caméra.
  5. L'application ou le lecteur de code QR détectera automatiquement le QR code et affichera les informations qu'il contient.
  6. Les informations affichées peuvent varier en fonction du contenu du QR code. Dans votre cas, le QR code contient des informations sur l'état émotionnel, l'enregistrement local des données et le suivi des émotions sur une période d'un mois.
